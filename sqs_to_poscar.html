The Special Quasirandom Structure (SQS) technique allows us to model the disordered behavior of alloys. One tool that can be used for this purpose is the Alloy Theoretic Automated Toolkit (ATAT), which generates an SQS file named "besqsqs.out". In order to perform atomistic simulation, this file needs to be converted into a readable format.

This small program can help with this task by converting the "besqsqs.out" file generated from ATAT into the "POSCAR" file format, which can be used for VASP calculations. To use this feature, simply copy and paste the "besqsqs.out" file and click the "convert" button to generate the "POSCAR" file on the right.
